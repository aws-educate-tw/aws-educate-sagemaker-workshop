問卷中的服務介紹
"""
EC2 (Elastic Compute Cloud)：
EC2 是一種提供可擴展計算能力的服務。用戶可以在雲中快速啟動虛擬機器，選擇不同的作業系統和配置。這對於處理不同的工作負載，例如應用程式伺服器、資料庫或後端伺服器等都非常有用。

S3 (Simple Storage Service)：
S3 是一種物件存儲服務，提供安全、耐用和高度擴展的存儲解決方案。它常用於備份和恢復、資料檔案存儲、網站內容等。用戶可以從任何地點存取存儲在S3上的資料。

Lambda：
Lambda 是一種無伺服器計算服務，允許用戶運行代碼而無需管理伺服器。Lambda 可以自動擴展，只在代碼執行時收費。這種服務非常適合自動化任務和響應事件，如檔案上傳到S3或更新至DynamoDB。

IAM (Identity and Access Management)：
IAM 是一種管理訪問AWS服務和資源的安全方式。它可以幫助用戶控制誰可以進行哪些操作，管理使用者、群組、權限和角色。

CloudWatch：
CloudWatch 是一種監控服務，用於監控AWS雲資源和運行在AWS上的應用程式。它可以用來收集和追蹤指標、收集和監控日誌檔案，並設置警報。

DynamoDB：
DynamoDB 是一種快速且靈活的NoSQL資料庫服務，適用於所有規模的應用程序。它支持文檔和鍵值資料模型，並保證單毫秒級的延遲性能。

API Gateway：
API Gateway 是一種讓開發者輕鬆部署、維護、監控和保護API的服務。它處理所有的接收調用，包括流量管理、授權和訪問控制、監控以及API版本管理。

ELB (Elastic Load Balancing)：
ELB 自動分配進入應用程式的流量，提高應用程式的可用性。它可以在多個EC2實例間平衡負載，確保分配適當的資源以處理各種流量模式和疲勞點。
